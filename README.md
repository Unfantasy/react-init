# react-s
我的reactPC端标准工程 (react + antd + dva)

## How to use

1. clone
```
$ git clone git@github.com:Unfantasy/react-init.git
```
2. set this project to yourself
```
$ cd react-s
$ rd /s/q .git
$ git init
```
3. install package
```
$ cnpm i
```
4. development
```
$ npm start
```
5. build
```
$ npm run build
```
